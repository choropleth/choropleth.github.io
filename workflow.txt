Минутка картографии:
Для отображения на нашей карте субъектов России нам необходимы данные об их границах.
1. Скачиваем на gislab.ru генерализованные слои границ субъектов РФ http://gis-lab.info/qa/rusbounds-rosreestr-gen.html
с порогом 10000 метров (достаточная точность для картограммы). 

2. Эти данные представлены в виде shape-файлов в проекции Альберс-Сибирь. 
Соответственно, сначала нужно преобразовать shp-файл по стандарту WGS-84(EPSG:4326).
    
Преобразовывать будем с помощью утилиты ogr2ogr из комплекта поставки GDAL.
Создадим файл проекции albres.prj:

+proj=aea +lat_1=52 +lat_2=64 +lat_0=0 +lon_0=105 +x_0=18500000 +y_0=0 +ellps=WGS84 +units=m +towgs84=28,-130,-95,0,0,0,0 +no_defs

Запустим преобразование:

> ogr2ogr -f 'ESRI Shapefile' -s_srs albres.prj -t_srs EPSG:4326 ./fixed.shp ./regions2010_sib_10.shp
    
3. Теперь нам необходимо преобразовать полученный файл в формат GeoJSON для последующего использования в \
LeafletJS. Используем опять же ogr2ogr:

> ogr2ogr -f 'GeoJSON' rusbounds_2010.json fixed.shp

Получаем GeoJSON файл, готовый к использованию в leaflet.
